# aws-ec2-linux-learning
Hands-on learning repository for AWS EC2, Ubuntu, and Amazon Linux, including Linux commands, SSH connectivity, Key pair, and screenshots.

## Overview

This repository documents my practical learning and hands-on experience with AWS EC2 instances using Ubuntu and Amazon Linux.

## Topics Covered

- AWS EC2 Instance Creation through
  -Ubuntu
  -Amazon Linux
- SSH Connectivity
- Key Pair Creation 
- Linux Commands
- User Permissions and sudo
- Screenshots and Command Outputs

## Learning Objectives

- Understand AWS EC2 fundamentals.
- Launch and manage EC2 instances.
- Connect to Linux servers using SSH.
- Practice common Linux commands.
- Learn the purpose and usage of sudo.
- Document learning with screenshots and notes.
