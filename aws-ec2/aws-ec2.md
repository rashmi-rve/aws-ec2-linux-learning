# What is AWS?
<img width="1907" height="906" alt="AWS-console-home" src="https://github.com/user-attachments/assets/324e8455-dcc9-4b02-aef8-56ce17479763" />





Amazon Web Services (AWS) is a cloud computing platform provided by Amazon that offers a wide range of on-demand services, including computing power, storage, databases, networking, security, analytics, artificial intelligence, and application development tools. Instead of purchasing and maintaining physical servers, organizations can use AWS to access computing resources over the internet and pay only for what they use.


Amazon Elastic Compute Cloud (EC2) is a web service provided by Amazon Web Services (AWS) that allows users to create and manage virtual servers in the cloud.

**EC2 enables users to:**
- Launch virtual machines called instances.
- Choose different operating systems such as Ubuntu, Amazon Linux, and Windows.
- Scale computing resources based on requirements.
- Connect securely using SSH (Linux) or RDP (Windows).
- Run applications without purchasing physical hardware.
  

### Key Features of EC2

- On-demand virtual servers
- Multiple operating system options
- Flexible instance types (CPU, Memory, Storage)
- Secure access using key pairs and security groups
- Easy scaling and monitoring
- Pay-as-you-go pricing


### Why Use EC2?

- Eliminates the need for physical servers.
- Provides quick server deployment.
- Supports development, testing, and production environments.
- Enables learning and practicing Linux and cloud technologies.
